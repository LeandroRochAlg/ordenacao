# Algoritmos de Ordenação de vetores em C
Os algoritmos empregados são os seguintes:
- Bolha;
- Bolha Inteligente;
- Seleção;
- Inserção;
- MergeSort;
- QuickSort;
- Dual-Pivot QuickSort.

O programa também é capaz de criar arquivos com números aleatórios ou ordenados de forma crescente ou decrescente.
Além disso, um algoritmo de busca binária e de busca linear são implementadas.

Ao ordenar, este programa conta as métricas abaixo e as salva em um arquivo, bem como o vetor ordenado.
- Número de comparações de chaves;
- Número de cópias de registros (trocas);
- Tempo de execução (clock).
